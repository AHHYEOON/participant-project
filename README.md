# 지역별 참가자 데이터 분석 프로젝트

## 프로젝트 소개
CSV 데이터를 분석하여 지역별 참가자 분포를 시각화하는 프로젝트입니다.
pandas와 matplotlib를 활용하여 데이터 처리 → 집계 → 시각화 흐름을 구현했습니다.

---

## 프로젝트 목적
데이터를 분석하고 결과를 직관적으로 표현하는 과정을 통해
데이터 처리 및 시각화 구조를 이해하는 것을 목표로 했습니다.

---

## 데이터 설명
- region : 지역 정보
- count : 참가자 수

지역별 참가자 규모를 비교하기 위한 데이터입니다.

---

## 사용 기술
- Python
- pandas
- matplotlib

---

## 처리 과정
CSV 로드  
→ 조건 필터  
→ groupby 집계  
→ 그래프 시각화  

---

## 결과
지역별 참가자 수를 막대그래프로 표현하여
데이터 분포를 직관적으로 확인할 수 있습니다.

---

## 배운점
- pandas를 활용한 데이터 집계
- matplotlib 시각화
- 데이터 흐름 구조 이해
